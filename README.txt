ANIMAL GUESSING GAME 
	created by Rafael Calpena, Leobardo Garcia, and Armando Garcia-Olvera

INSTRUCTIONS:	

Tested and Completed with python 2.7.x

install the following files into the same directory:

-ai.py
-animals.py
-questions.py

open the terminal or command line prompt and set path to the directory which contains the three mentioned files.

type "python ai.py" and begin playing the game.
===================================================
Type "y" for yes and "n" for no when deciding the best answer to the questioned asked. Continue to answer the questions until the agent guesses your animal. Follow the on-screen instructions to finish gameplay.
